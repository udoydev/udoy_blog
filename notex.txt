Topic: Django Custom User Model

Problem
Need roles (admin, moderator, user) and flexible auth system.

Concept
Default Django uses auth_user table.
To extend user features, create a custom model using AbstractUser.

Example
class User(AbstractUser):
    role = models.CharField(max_length=20)

Rule
Custom user model must be defined BEFORE first migration.
Otherwise resetting DB or complex migrations are required.
